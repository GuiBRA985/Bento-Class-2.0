# BENTO HANDOFF 02

**Project:** Bento Class 2.0

## Vision

Bento Class is more than an English course. Bento is a teacher with
personality.

-   Serious while teaching.
-   Ironic only at the end.
-   Never offensive.
-   Always encouraging.

**Rule:** Teach first. Joke later.

## Lesson Flow

Greeting: - Motivational message. - Ends with **Keep on.**

Stages: 1. **Keep listening.** (Vocabulary) 2. **Keep writing.**
(Handwriting) 3. **Keep speaking.** (Pronunciation) 4. **Keep trying.**
(Bento Lab)

## Pronunciation

-   Highest priority.
-   Evaluated by AI.
-   Real feedback.

## Handwriting

Purpose: - Create muscle memory. - Never become a barrier.

OCR should only verify: - Enough handwriting. - Numbered sentences. - No
obscene/inappropriate content.

Result: - Bento Irony Index.

## Bento Irony

Humor about the situation, never about the student.

Examples: - I've seen doctors with better handwriting. - The archaeology
department is still translating your manuscript. - Good news. We
identified three words without carbon dating.

Always finish with encouragement.
